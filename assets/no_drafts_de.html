Gratulation, Sie haben die App erfolgreich konfiguriert und können nun mit dem SMS-Versand beginnen.

Bitte nehmen Sie sich einen Moment Zeit um die Funktionsweise der App zu überblicken:

  • Eigene Absenderadresse definieren
    Sie können für jede SMS die Sie verschicken eine eigene Absenderadresse definieren. Um Ihre Absenderadresse nicht jedes Mal neu eingeben zu müssen können Sie in den Einstellungen eine Standardadresse angeben. Wollen Sie einzelne SMS lieber mit einer anderen Absenderadresse verschicken, dann steht es Ihnen jederzeit frei Ihre selbst gewählte Standardadresse zu überschreiben. Sie bekommen diese Möglichkeit bei jeder SMS separat. Probieren Sie es einfach aus!
    Bitte beachten Sie, dass in der Route Basic keine eigenen Absenderadressen möglich sind.
  • SMS zeitversetzt versenden
    Sie können einen Zeitpunkt definieren an dem die SMS verschickt werden soll. Falls Sie den zeitversetzten Versand aktivieren, wird Ihre SMS an die Server übertragen und dort zwischengespeichert. Zum angegebenen Zeitpunkt wird die SMS dann verschickt. Sie müssen also zu dem angegebenen Zeitpunkt nicht online sein, das erledigen die Server für Sie.
  • Entwürfe speichern
    Falls Sie einen Text schreiben und keine Verbindung zum Internet besteht, so dass die SMS nicht verschickt werden kann, können Sie trotzdem gelassen bleiben. Mit dem Zurück-Button können Sie Ihre aktuelle Version als Entwurf speichern und zu einem späteren Zeitpunkt verschicken.
  • Verlauf einsehen
    Sie können jederzeit einsehen welche SMS Sie schon verschickt haben. Dazu wählen Sie auf der Startseite Menü -> Verlauf
