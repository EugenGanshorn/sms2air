Welcome to sms2air!

Looks like you're using sms2air the first time.

With this App you can send SMS over the Internet via the smstrade.de gateway.
To do so, click on the button below and enter your personal gateway key. You'll find that key on http://smstrade.de in the section Interfaces - HTTP(s) interface.

After you're done, you can start to send SMS with this App.

Have fun! :)

Please notice: This is no official App from smstrade.de. Therefore you should contact the author of this App directly if you have any support issues relating to this app. You'll find the contact information in the android market.
